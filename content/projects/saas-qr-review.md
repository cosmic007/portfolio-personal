# QR-Based Review Collection SaaS

A SaaS platform that enables hotels and restaurants to effortlessly collect customer reviews through QR codes, improving customer feedback visibility and online reputation management.

## Overview

This team project (5 members) was designed to solve a critical problem in the hospitality industry: making it easy for customers to leave reviews and for businesses to manage their online reputation in real-time.

## Features

- **Dynamic QR Code Generation** - Generate unique QR codes for each location or service
- **Mobile-Friendly Review Form** - Easy-to-use review collection interface
- **Real-time Dashboard** - Track feedback as it comes in
- **Analytics & Insights** - Identify trends and common feedback themes
- **Multi-location Support** - Manage multiple business locations
- **Integration with Review Platforms** - Push reviews to Google, TripAdvisor, etc.
- **Customer Engagement Metrics** - Monitor response rates and trends

## Technology Stack

- **Frontend**: React/TypeScript
- **Backend**: Node.js / Express
- **Database**: PostgreSQL
- **QR Library**: QR Code generation libraries
- **Deployment**: Cloud platform (AWS/Vercel)

## Key Accomplishments

- Designed and implemented complete QR code flow for review collection
- Built responsive UI for mobile devices
- Created real-time analytics dashboard
- Implemented secure data management
- Developed integration APIs for third-party review platforms
- Established CI/CD pipeline for continuous deployment
